## ‚ú® Why This Extension?

Nowadays, users often open **two or more side-by-side ChatGPT windows** to cross-reference answers or take notes.  
This workflow has a few issues:

- ‚ùå **Context is lost**: You can‚Äôt automatically link your questions and answers across multiple tabs.  
- ‚ùå **Screen clutter**: Multiple windows consume valuable screen space, making it hard to focus.  
- ‚ùå **Messy workflow**: Switching back and forth reduces efficiency.  

**Disposable ChatGPT Popup** solves this problem by giving you a **temporary, draggable popup** that:

- Opens right on the page you are reading  
- Keeps previous ChatGPT context intact  
- Disappears once closed  
- **Does not require API keys**  
- **Is completely free ‚Äî no subscription or payment needed**

## üõ†Ô∏è Installation

### From Chrome or Edge (Developer Mode)

1. Download zip file from releases.
2. Extract it in your file system
3. Open Chrom and navigate to `chrome://extensions/`
4. Enable Developer Mode (top right)

5. Click Load unpacked and select the folder containing this repo

6. Pin the extension if needed


## Usage 
1. Go to your ChatGPT
2. Highlight the text you want explained while holding the Alt key

3. A popup will appear with a temporary ChatGPT session

4. Drag, resize, and close as needed
